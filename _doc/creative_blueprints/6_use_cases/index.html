Key Use Cases

Two modes, many ways to play

💬

Companion chat

User taps a character and chats 1:1 — visual, in-character, remembered. The companion pings when away. The primary, mass-market door.

Chat Mode · primary
🃏

Character collecting

User browses and switches between many companions like a gacha gallery — discovering and bonding with new characters from the feed.

Chat Mode · collect
📖

Read-along

User rarely chats — treats Story Mode as an evolving novel that writes itself, checking in to read new beats as they land.

Story Mode · low intervention
✍️

Co-author

User intervenes often, steering plot and dialogue while the engine fills gaps. Scenario "what if" prompts ripple into upcoming beats.

Story Mode · high intervention
