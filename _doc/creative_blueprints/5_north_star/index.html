Verified Operation

Story Beat Generated — a scene, narrative event, or chapter produced by the autonomous engine (or triggered by a user message), delivered to an active user in the deployed app. Clicks, logins, and page views are not operations.

WVO Formula

Beats per day per user 3
× Days per week 7

× Active users Assumption — adjust this
